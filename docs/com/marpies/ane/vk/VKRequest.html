parametersproperty 
parameters:Object

Request parameters, a key-value object. Can be null if the request method does not require any parameters.

The value can be one of the following types:

  • int, uint, Number
  • String
  • Array (of Strings)
  • Vector.<String>
If one of the values in an Array must be numeric then simply wrap it in quotes.

An example parameters value for users.get request: 
         VK.request
             .setMethod( "users.get" )
             .setParameters( {
                 user_ids: ["210700286"],
                 fields  : "photo_200,city,verified"
             } )
             .setResponseCallback( onUsersGetResponse )
             .send();
